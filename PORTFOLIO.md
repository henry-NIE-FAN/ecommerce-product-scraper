# Upwork Portfolio Text

## Title

E-commerce Product Data Scraper with CSV/JSON Export

## Short Description

I built a Python scraper that collects product information from an e-commerce catalog and exports clean, spreadsheet-ready CSV or JSON files.

## What It Does

- Scrapes product titles, prices, ratings, availability, and product links
- Supports multiple listing pages
- Exports results to CSV or JSON
- Includes an offline demo mode for reliable testing
- Uses polite request headers and a configurable delay between pages

## Tools Used

- Python
- Standard library HTML parser
- CSV and JSON export
- Unit testing

## Client Value

This type of script can help clients collect product data, monitor competitor catalogs, build lead lists, prepare market research spreadsheets, or automate repetitive copy-and-paste work.

## Suggested Upwork Portfolio Summary

This project demonstrates a practical product data extraction workflow using Python. The scraper visits e-commerce listing pages, extracts key product fields, normalizes the data, and saves it into CSV or JSON files that clients can open in Excel or Google Sheets.

I designed it with a clean command-line interface, testable parsing logic, and sample output so the workflow is easy to review and adapt for real client requirements.
